library(ggplot2)
library(palmerpenguins)

data("penguins")

# 1) ggplot(data = penguins): Em ggplot2, um grfico  iniciado com a funo ggplot().
# Ela cria um sistema de coordenadas ao qual  possvel acrescentar camadas. O 
# primeiro argumento da funo ggplot()  o conjunto de dados a ser usado no 
# grfico. Neste caso,  penguins.

# 2) +: Depois, acrescenta-se um smbolo + para adicionar uma nova camada ao seu
#grfico. O grfico  finalizado adicionando uma ou mais camadas ao ggplot().

# 3) geom_point(): Em seguida, deve-se escolher um geom adicionando uma funo geom. 
# A funo geom_point() utiliza pontos para criar grficos de disperso, a funo 
# geom_bar usa barras para criar grficos de barras, e assim por diante. Neste caso, 
# escolha funo geom_point para criar um grfico de disperso de pontos. O pacote 
# ggplot2 vem com muitas funes geom diferentes. Voc aprender mais sobre geoms 
# posteriormente no curso.

# 4) (mapping = aes(x = flipper_length_mm, y = body_mass_g)): Cada funo geom em 
# ggplot2 leva um argumento de mapeamento. Com isso, define-se como as variveis do
# seu conjunto de dados sero mapeadas de acordo com as propriedades visuais. O 
# argumento de mapeamento sempre  combinado com a funo aes(). Os argumentos x e
# y da funo aes() especificam quais variveis devem ser mapeadas no eixo X e no 
# eixo Y do sistema de coordenadas. Neste caso, deseja-se mapear a varivel 
# flipper_length_mm no eixo X e a varivel body_mass_g no eixo Y.

ggplot(
  data = penguins, 
  mapping = aes(
    x = flipper_length_mm, 
    y = body_mass_g,
    colour = species,
    shape = species,
    # size = species
    )
  ) + 
  geom_point() + 
  
  # geom_smooth: com essa funo  possvel mostrar linhas de tendncia
  geom_smooth(method="gam", 
              formula = y ~s(x)) +
  
  # labs(): Para mudar/definir os rtulos
  labs(y = "Peso (em gramas)", x = "Tamanho (em milmetros)", colour = 'Espcie') + 
  
  # facet_grid: divide os quadros para cada um dos elementos da varivel especificada
  facet_grid(facets="species")
